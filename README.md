# Developer: Setup & Install

1. Fork repo

2. pip install pipenv

3. pipenv shell

4. pipenv install pip

5. export FLASK_APP=web_app

6. export FLASK_ENV=developer

7. flask run

Should now be running on http://127.0.0.1:5000/
